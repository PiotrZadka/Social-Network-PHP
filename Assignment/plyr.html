Plyr

Website:https://plyr.io/

GitHub:https://github.com/sampotts/plyr

Lightweight, accessible and customizable media player. As simple as previously mentioned Afterglow.

Key features:

  • Lightweight (10kb)
  • Available support on GitHub
  • Free
  • Used by known companies such as Peugeot, TomTom
  • Supports CDN
  • Supports all browsers
  • Supports youtube links and IDâ€™s
  • Possible to implement other video formats
  • Highly customisable interface
  • Possible to contribute on GitHub and frequently updated

How to use:

Guide: http://afterglowplayer.com/wizard

Plyr extends upon the standard HTML5 markup so that's all you need for those types.

More info on advanced HTML markup can be found under initialising.

Include two links in header:

<script src="https://cdn.plyr.io/2.0.15/plyr.js"></script>
<link rel="stylesheet" href="https://cdn.plyr.io/2.0.15/plyr.css">

Include these two lines of code in document:

<div id='plyr-youtube' data-type="youtube" data-video-id="NlezRkAubys"></div>
<script>plyr.setup("#plyr-youtube")</script>
