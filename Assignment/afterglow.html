Afterglow

Website:http://afterglowplayer.com/

GitHub:https://github.com/moay/afterglow

Afterglow is a tool to create fully responsive and totally awesome video players from HTML5 video elements with as little effort as possible.

It’s super simple to use and comes loaded with features.

Key features:

  • Easy to use and user friendly,
  • Can support video format such as vimeo, youtube or local files
  • Works with all major browsers and devices
  • Fully responsive,
  • Supports CDN
  • Has a community forum,
  • Possible to contribute on GitHub and frequently updated
  • Documentation fully explained and hands on examples
  • Relies on popular scripts such as video.js
  • Free!

How to use:

Guide: http://afterglowplayer.com/wizard

Afterglow website contains a simple wizard feature that generates specific code for what kind of video format we would like to use on our website.

Therefore, example of YouTube links:

<script type="text/javascript" src="//cdn.jsdelivr.net/npm/afterglowplayer@1.x"></script>
<video class="afterglow" id="my-video" width="500" height="300" data-youtube-id="https://www.youtube.com/watch?v=G3Fc_eOkM8M&t=0s"></video>

This parameter"data-youtube-id" can be later targeted in our global feed to store any link from the user input and stored later in our database in any format to later retrieve and displayed in video player.

Data-youtube-id can also handle short ID such as “G3Fc_eOkM8M&t” instead of full link.


User can post a link in the global feed and from there we can decide if we store full link or just the ID of the video link in our database.

Once stored we can retrieve it back to our global feed and show the user his link well presented in Afterglow Video Player.
